The IASBS Thesis document class offers the following options:

  o  Master's thesis (final mode)
  o  Master's thesis (review mode)
  o  PhD proposal
  o  PhD dissertation (final mode)
  o  PhD dissertation (review mode)
  o  Bachelor's project report (final mode)
  o  Bachelor's project report (review mode)

A sample template is available for the document style, which can be compiled in LaTeX based
on any of these options. The compiled results are provided in this folder.
